## Getting Started



### Prerequisites

The only thing needed is Python 

### Usage

1. Clone the repo
   ```sh
   git clone https://github.com/HelplessSneeker/static_site.git
   ```
2. For running the Site Locally execute the main.sh file
3. for building execute the build.sh file

To change the generated HTML just put your desired Markdown files inside the content folder the folder Structure will be reflected in the Routing

Static content like images and Stylesheet need to be provided in the static folder

For changing the Core Structure of the HTML the template.html file can be changed
